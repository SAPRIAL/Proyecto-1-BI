Modelo Multinomial NB

Se utilizó un algoritmo de clasificación multinomial bayesiano y un data set de comentarios para construir un modelo que permitiera clasificar comentarios como buenos, malos o neutros a partir de su texto.

El algoritmo Multinomial NB (Algoritmo multinomial bayesiano ingenuo) es un algoritmo de clasificación supervisado que comúnmente se utiliza para procesamiento de lenguaje natural. Basado en el teorema de Bayes, asume que todas las palabras en el texto a analizar son independientes, es decir, que la ocurrencia o no de una palabra en el texto no está relacionada con las demás.
Se basa en la distribución multinomial y a partir del número de ocurrencia de las palabras, el algoritmo calcula la probabilidad de que una cadena de texto pertenezca a una determinada categoría.

Otros algoritmos evaluados

También se utilizaron los algoritmos "linear SVC" y "random forest" para la tarea de clasificación. Sin embargo, el algoritmo que mejor rendimiento mostró (el que mejor logra predecir la categoría de un comentario) es el "multinomial NB". Por este motivo, se seleccionó dicho algoritmo para el modelo.

Rendimiento del modelo

El modelo tiene las siguientes métricas

Accuracy F1 score
84.81% 0.6547

La métrica "accuracy" hace referencia a que tan bien el modelo puede predecir la categoría de un comentario. Un accuracy del 84% quiere decir que de 100 comentarios, el modelo clasifica bien 84.

La métrica "F1 score" es un indicador comunmente utilizado en la evaluación de modelos de clasificación. Varía entre 0 y 1. Un valor de 0.65 se considera aceptable.

Métricas por categoría

El modelo presenta las siguientes métricas por categoría.

Categoría F1 score
Malo 0.77
Bueno 0.93
Neutro 0.27

El modelo presenta valores buenos de f1 socre para las categorías bueno y malo. Sin embargo, el f1 score de la categoría neutro no es aceptable. Esto quiere decir que el modelo no es bueno para clasificar comentarios neutros. Usted deberá sospechar de un comentario clasificado como neutro, y revisar si dicha clasificación tiene sentido.
