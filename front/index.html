HOTELS REVIEWS

Esta herramienta permite predecir si un comentario sobre un hotel es positivo, negativo o neutro.

Para obtener más información sobre el modelo utilizado diríjase a Sobre el modelo

El comentario es: neutro

Atención! El modelo no es bueno prediciendo comentarios neutros. Vea "¿Cómo interpretar el resultado?" para más infromación.

Este modelo solo predice si un comentario es bueno, malo o netro. Así, no debes interpretar el resultado como una verdad absoluta. Aunque el modelo tiene una presición del 84% aproximadamente, no es igual de bueno clasificando comentarios en todas las categorías. En otras palabras, el modelo es relativamente acertado clasificando comentarios buenos y malos, pero no es tan bueno clasificando comentarios neutros. Puedes encontrar las métricas del modelo junto con una explicación de qué representan en sobre el modelo.
